	二分查找
	当中间元素大于其相邻的后续元素时
	说明中间元素左侧（包含中间元素）必然包含一个局部最大值
	这时，中间元素也可能是峰值点，所以移动时，end = mid
	而相反情况，中间元素小于其向相邻后续元素
	则中间元素右侧比包含一个局部最大值，
	这时候中间元素一定不是局部最大点，start = mid + 1
